# Helpdesk
Ferramenta interna de suporte para operadores de ISP. Desenvolvida com Node.js e Express, guia os atendentes por fluxos estruturados para problemas comuns como conexão lenta, mudança de endereço e serviços SVA. 
O Web Service facilita o uso de quem precisa de praticidade durante o atendimento, pois ao invés de anotar num bloco de notas e especificar cada coisa, o Web Service automatiza e facilita a vida do usuário, fazendo assim, o cliente que está sendo atendido ser respondido mais rapidamente. 

## URL do Web Service: https://helpdesk-izcw.onrender.com
